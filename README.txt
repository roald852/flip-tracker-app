Flip Tracker App

How to deploy on Vercel:
1. Download and unzip this folder.
2. Go to vercel.com and sign in.
3. Click Add New -> Project.
4. Either import from GitHub or use Vercel's upload option if shown.
5. If using GitHub, upload these files to a new repo first.
6. Framework preset: Vite
7. Build command: npm run build
8. Output directory: dist

If a link says 'deployment not found', it means there is no successful deployment yet or the URL is wrong.
